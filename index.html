HackTheBox - Lame (Linux Machine)

"Lame" is a Linux machine challenge released in 2017, that requires one exploit to gain root access. Upon enumerating all open ports on the network with Nmap, I found that an outdated and vulnerable version of Samba was running on port 445. After researching, I found that a remote code execution vulnerability CVE-2007-2447 can be used to gain root access. After gaining root access, I quickly found the flags to successfully complete this challenge.
